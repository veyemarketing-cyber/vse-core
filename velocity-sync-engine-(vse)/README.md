## Governance-as-Code
The VSE operates under a strict **Human-on-the-loop** philosophy. All autonomous actions are validated against the `vse_governance_manifest.yaml`, which enforces:
* **2-Agent-Minimum Consensus:** No action is taken unless the Synthesis and Reasoning agents agree.
* **Financial Guardrails:** Hard caps on API spend ($5.00/day) and bid adjustments (15%).
* **Artisanal Tone:** Automatic filtering of generic phrasing like "cutting-edge" or "in today's competitive landscape."
